📊 Sales Report Dashboard (Excel)
Overview

The Sales Report Dashboard is an interactive Excel dashboard designed to analyze sales performance across different regions, states, products, and time periods. It provides a comprehensive overview of key business metrics using dynamic charts, KPIs, and slicers, helping users monitor sales trends and make data-driven decisions.

Key Features
Interactive dashboard built entirely in Microsoft Excel.
Dynamic KPI cards displaying:
Total Units Sold
Total Sales Revenue
Total Orders
Total Customers
Monthly sales trend analysis.
State-wise sales comparison.
Region-wise sales distribution.
Product-wise sales analysis.
Order count versus sales comparison.
Interactive slicers for:
Ship Date
Order Date
Region
State
Product Name
Dashboard Insights
Monitor overall business performance using KPI cards.
Compare sales performance across different states and regions.
Identify top-performing products.
Track monthly sales growth and order trends.
Filter data dynamically to analyze specific regions, products, or time periods.
